# Snap_Eat_Food_Delivery
This is a full-stack food delivery web application inspired by platforms like Swiggy and Zomato. The application allows users to explore restaurants, view menus, add food items to a cart, place orders, and receive order confirmations. It is built using Java, JSP, Servlets, MySQL, HTML, CSS, and clean coding practices.
